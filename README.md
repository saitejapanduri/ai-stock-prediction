# AI Hybrid Stock Prediction System

A multi-model ensemble system for predicting Indian stock prices, combining LSTM, Random Forest, and Linear Regression models with a real-time data pipeline, news sentiment analysis, and risk/trade-setup logic — served through a Streamlit dashboard.

## Features

- **Hybrid prediction**: Weighted ensemble of LSTM (sequence model) and Random Forest (return-based) predictions
- **Technical indicators**: MA20, MA50, RSI, volatility, support/resistance, ATR
- **Trade setup generation**: Entry, stop-loss, and target price based on ATR and risk-reward ratio
- **News sentiment**: Pulls recent headlines via NewsAPI and scores sentiment with TextBlob
- **Risk analysis**: Volatility-based risk classification (Low / Medium / High)
- **Live data**: Pulled directly from Yahoo Finance via `yfinance`

## Project Structure

```
.
├── app.py                  # Streamlit dashboard and prediction pipeline
├── train_models.py         # Trains and saves RF, LSTM, and open-price models per stock
├── risk_analysis.py        # Volatility-based risk scoring
├── news_sentiment.py       # NewsAPI + TextBlob sentiment scoring
├── requirements.txt
├── .env.example
└── models/                 # Generated by train_models.py (not tracked in git)
```

## Setup

1. Clone the repo and create a virtual environment:
   ```bash
   git clone https://github.com/saitejapanduri/ai-stock-prediction.git
   cd ai-stock-prediction
   python -m venv venv
   source venv/bin/activate   # Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   python -m textblob.download_corpora
   ```

3. Set up your environment variables:
   ```bash
   cp .env.example .env
   ```
   Then edit `.env` and add your [NewsAPI](https://newsapi.org/) key:
   ```
   NEWSAPI_KEY=your_actual_key
   ```

4. Train the models (this creates the `models/` directory):
   ```bash
   python train_models.py
   ```

5. Run the app:
   ```bash
   streamlit run app.py
   ```

## Supported Stocks

The dashboard ships with four NSE stocks configured by default:

- RELIANCE.NS
- TCS.NS
- INFY.NS
- HDFCBANK.NS

This list isn't fixed — the system works with any valid Yahoo Finance ticker. To add a new stock:

1. Add its ticker to the `stocks` list in `train_models.py` and run it to generate that stock's models:
   ```python
   stocks = ["RELIANCE.NS", "TCS.NS", "INFY.NS", "HDFCBANK.NS", "NEWSTOCK.NS"]
   ```
2. Add the same ticker to the `st.selectbox` options list in `app.py`.
3. If you want sentiment analysis to return relevant news, add a ticker → company name mapping in `TICKER_TO_NAME` inside `news_sentiment.py`.

Any ticker recognized by `yfinance` will work — NSE (`.NS`), BSE (`.BO`), or US tickers like `AAPL`.

## Notes

- Models are trained per-stock and saved as `.pkl` (RF, open-price LR, scaler) and `.weights.h5` (LSTM) files in `models/`. These are excluded from version control — run `train_models.py` to regenerate them locally.
- Sentiment scores depend on NewsAPI's free-tier rate limits; if no articles are found or the API errors out, sentiment defaults to neutral (0).
